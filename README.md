# Enterprise Commerce Backend

Production-ready backend built using Python microservices.

## Overview

This project is a scalable e-commerce backend designed using a microservices architecture. It is being developed step by step following real-world software engineering practices.

## Planned Features

- User Authentication
- Product Management
- Order Management
- Notifications
- Analytics
- Docker Deployment
- Kubernetes Orchestration
- CI/CD Pipeline
- AWS Deployment
